Livepeer Video Gateway

Portal

Transcode video on the Livepeer network.

VOD ABR ladder transcoding and live RTMP→HLS streaming. Free during beta.

# VOD ABR ladder
curl https://api.example.com/v1/abr \
  -H "Authorization: Bearer sk-..." \
  -H "Content-Type: application/json" \
  -d '{"input_url":"https://example.com/source.mp4"}'

# Live RTMP→HLS
curl https://api.example.com/v1/live \
  -H "Authorization: Bearer sk-..." \
  -X POST

Request access

Sign up to the waitlist. We email a verification link; once verified, an admin reviews your signup and emails an API key.

API surface

  • POST /v1/abr — VOD ABR ladder transcode
  • POST /v1/abr/upload-url — presigned RustFS PUT for VOD ingest
  • POST /v1/live — allocate an RTMP ingest + HLS egress session
  • GET /v1/live/:id — live session status
  • DELETE /v1/live/:id — close a live session
  • GET /v1/capabilities — registry-backed transcode catalog

Backed by the Livepeer decentralized GPU network.
